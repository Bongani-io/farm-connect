# Farm-Connect Ecommerce

Farm-Connect is a modern e-commerce platform designed to bridge the gap between farmers and consumers. Built with Next.js, it provides a seamless shopping experience for customers while offering powerful management tools for farmers and administrators.

## Features

- 🛍️ Modern e-commerce frontend for customers
- 📊 Comprehensive back-office dashboard
- 📈 Real-time sales analytics and reporting
- 👥 Customer management system
- 🏪 Online store management
- 👨‍🌾 Farmer management tools
- 📦 Order processing and tracking
- 👥 Staff management capabilities

## Tech Stack

- **Framework:** Next.js 15.1.7
- **UI Library:** React 19
- **Styling:** Tailwind CSS + SCSS
- **Charts:** Chart.js
- **Icons:** Lucide React
- **Development:** ESLint, PostCSS

## Getting Started

1. Clone the repository:
```bash
git clone [your-repository-url]
cd farm-connect
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app directory
  - `/(front-end)` - Customer-facing store
  - `/(back-office)` - Admin dashboard
- `/components` - Reusable React components
- `/styles` - Global styles and Tailwind configuration
- `/public` - Static assets

## Development

The project uses modern development tools and practices:

- ESLint for code linting
- Tailwind CSS for styling
- SCSS for custom styles
- Chart.js for data visualization
